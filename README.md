# String Operations Exercise

Welcome to this exercise about strings operations.
If you need help running the tests or submitting your code, check out `HELP.md`.

## Instructions

Your task is to implement these string operations:
* Get the string length
* Copy a string into another
* Compare two strings

### Note
* Using the `<string.h>` library is NOT allowed
